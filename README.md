# Vitralis — 3D Window Configurator Template

A clean, interactive HTML template for window and glazing businesses. Includes a real-time 3D configurator, pricing calculator, glazing cross-section viewer, and gallery layout — no build step required.

## Demo

[Live preview](https://kirindenis.github.io/owlos-template-win/) <!-- enable GitHub Pages on main branch / root -->

## Features

- Interactive 3D window and glazing cross-section (Three.js, self-hosted)
- Real-time price calculator (profile depth, glazing type, sections, color)
- Fully responsive — mobile-first layout
- Self-hosted fonts (Inter, DM Mono, Courier Prime) — no Google Fonts CDN
- No external CDN dependencies — 100% self-contained, works offline
- No analytics, tracking pixels, or advertising scripts
- Well-commented source code — easy to customise

## Quick Start

1. Clone or download this repository
2. Open `index.html` in any modern browser — no server required
3. Replace placeholder images in `assets/images/` with your own project photos
4. Edit prices, colors, and copy in `configurator.js` and `index.html`
5. Update branding (company name, phone, email) in the footer section of `index.html`
